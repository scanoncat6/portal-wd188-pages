# portal-wd188-pages

## 项目简介

`portal-wd188-pages` 是一个用于归档和发布多个独立 HTML 页面的仓库。本项目不针对任何特定域名或网站，仅作为通用页面存储与分发用途，方便不同场景下的页面展示与访问。

## 目录说明

```
portal-wd188-pages/
├── index.html          # 入口页面
├── pages/              # 存放各类独立 HTML 页面
│   ├── page-01.html
│   ├── page-02.html
│   └── ...
├── assets/             # 静态资源文件（CSS、JS、图片等）
│   ├── css/
│   ├── js/
│   └── images/
└── README.md           # 本文件
```

- **index.html**：仓库主入口页面，可提供导航或简要说明。
- **pages/**：存放所有独立 HTML 页面，按需命名与组织。
- **assets/**：存放页面所需的静态资源，如样式表、脚本、图片等。

## 页面归档说明

本仓库中的 HTML 页面为独立归档文件，不包含动态后端逻辑，可直接通过浏览器或静态托管服务访问。每个页面均保持独立性与完整性，方便单独使用或整合到其他项目中。

## 维护说明

- 如需添加新页面，请在 `pages/` 目录下创建对应的 `.html` 文件，并确保引用资源路径正确。
- 建议保持页面命名规范，避免使用特殊字符或空格。
- 若页面涉及外部资源（如字体、图标库），建议将资源文件一并放入 `assets/` 目录，减少外部依赖。
- 本仓库不提供自动部署或持续集成配置，使用者可根据需要自行配置托管服务。
